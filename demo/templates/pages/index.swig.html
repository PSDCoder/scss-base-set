Grid

  • Grid has 12 columns
  • Container class: .grid
  • Row class: grid__row
  • Cell class: grid__col_%breakpoint%-%size%
  • Offset class: grid__col_%breakpoint%-offset-%size%
  • Pull class: grid__col_%breakpoint%-pull-%size%
  • Push class: grid__col_%breakpoint%-push-%size%
  • Or if you need only for one breakpoint:
    • Offset class: grid__col_%breakpoint%-only-offset-%size%
    • Pull class: grid__col_%breakpoint%-only-pull-%size%
    • Push class: grid__col_%breakpoint%-only-push-%size%
