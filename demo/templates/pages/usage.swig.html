Usage

You can simply load dist css file from dist folder or you can build only necessary components.

For that you should include components in your main scss file like in /src/main.scss before your app code.

Also you can redefine variables. List of variables you can find in /src/_variables.scss. And modules can has own variables list based on global variables - you can find it in modules folder.
